# Amazon Product Inspector (Beta)

Chrome extension for fast Amazon product research.
Inspect listings, evaluate opportunity, save & compare products.

## Install (for testers)

1. Download this repo as ZIP
2. Unzip it
3. Open Chrome â†’ chrome://extensions
4. Enable **Developer mode**
5. Click **Load unpacked**
6. Select the unzipped folder

## Update
- Pull latest changes or re-download ZIP
- Click **Reload** in chrome://extensions

## Notes
- Some Amazon layouts may hide price
- Price ranges appear for variants
- Profit calculator is an estimate
